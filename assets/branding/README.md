# FarmGenius Legacy Logo Pack

This logo set was created to honor **Dr. Pascal Fidelis Mujuni** and represent the family’s commitment to running Dr. PF Farm Trust to his standards.

## Files

- `farmgenius_logo_primary.svg` — full-color primary brand lockup (preferred)
- `farmgenius_logo_mark.svg` — emblem-only mark (app icon/avatar/favicon/sign stamp)
- `farmgenius_logo_monochrome.svg` — single-color variant for uniforms, engraving, receipts, and one-ink printing

## Symbol Meaning

- **Outer ring**: continuity, trust, and long-term stewardship.
- **Rising sun**: hope, vision, and future generations.
- **Cultivated field bands**: practical farming discipline and productivity.
- **Central leaf form**: life, health, and science-led agriculture.
- **Wordmark subtitle**: explicit memorial recognition of Dr. PF legacy.

## Recommended Use

- Use `farmgenius_logo_primary.svg` on app splash, website, presentations, and signage.
- Use `farmgenius_logo_mark.svg` for compact spaces where text is too small.
- Use `farmgenius_logo_monochrome.svg` when only one color is available.

## Export Tips

- Keep SVG as source-of-truth.
- Export PNG sizes from SVG as needed (e.g., `1024`, `512`, `256`, `128`).
- For print, export PDF from SVG at 300 DPI equivalent.
